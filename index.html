    Customizing a Workbook

    To customize a workbook:

    1. To insert a formula, click Formula Wizard from the Analysis tab.

      1. Select the Environment from the drop-down list. All available environments in Web Client are listed. When set as Current Env., the formula will be updated to retrieve data from the selected environment.

        Note

        When you switch environments from the title bar, the data will be updated automatically everywhere EXCEPT for formulas created in specific environments.

      2. Select the Data Model, Field and Operator from their respective drop-down lists. You can also search the item by typing the name in the field for a quicker selection.

        • The Data Model is the data source that contains the field you wish to use in the formula. Data models are listed in alphabetical order.

        • The Field is the field you wish to use in the formula. Fields are listed in alphabetical order.

        • The Operator determines the type of aggregation that should be performed on the selected field.

          Note

          The Dynamic List operator creates a list that can be dynamically updated depending on the filters used.

      3. Click Add Filter to add a filter to the formula. The Add Group feature allows for grouping filters and transferring them to the newly created group by dragging the handle bar . Groups can be a part of other groups and configured between OR and AND. For formula examples, see Workbook Formula Examples.

        • Select the Filter field, operator and value.

          • The Field is the field you wish to use in the filter. Fields are listed in alphabetical order. You can use the prompt to select the field.

          • The Operator determines the type of operation that should be performed on the selected field.

          • The Value can be hardcoded or can also be a cell reference.

        • If you wish to reference a cell, click the underscore in the Value field and select the cell. You can also enter the value in the cell.

        • By default, when you enter a character value, the workbook assumes it to be a cell reference. To specify the value as text, click the icon and the color will change to blue .

        Note

        When selecting a character value using the prompt, the value is automatically set to text.

      4. You can click Preview Value to preview the formula result. If the formula is incorrect, a message will be displayed. When resolving the formula, a timeout occurs when the query takes 300 seconds (5 minutes) or when the data results retrieved for the formula exceed 500,000 entries.

      5. Click Save.

      Note

      If the content of a referenced cell is modified, all formulas referencing the cell will be automatically recalculated.

      The Formula Wizard generates a function in the workbook that is visible in the formula bar. When clicking the formula bar, the NECFORMULA function (which is similar to the NECACCESS function in the Excel Add-in) appears in a pop-up window.

      Note

      As opposed to Excel Add-in, you do not need to create a reference configuration. Thanks to smart caching, configuration is done automatically based on the formula created.

    2. When using more than one NECFORMULA within a cell, the Formula Wizard allows you to assign names to each formula.

      1. Select a cell with multiple NECFORMULA, then click Formula Wizard from the Analysis tab.

      2. In the list on the left, click the to rename the NECFORMULA of your choice.

      3. Once you save the changes, the custom names are added to the NECFORMULA in the cell.

      Note

      This feature only appears when there are multiple formulas within a single cell.

    3. If you wish to use a global variable or a global selection page variable, click Functions from the Analysis tab.

      1. Select a variable from the Global Variable or Global Selection Page Parameters list.

      2. Click Save. The variable can be used in a formula provided that its cell is referenced.

      Note

      Global variables use @@ in front of the parameter. Selection Pages use ## in front of the parameter. Reports can use both types of parameters (## and @@). To view a selection page example, click here.

    4. When finished, click Save to keep the workbook changes.
